# Threat Model

## Adversaries
- State-level censors blocking access to news and communication
- Surveillance actors monitoring user activity
- Malicious intermediaries attempting traffic analysis

## Risks
- Network traffic interception
- Metadata leakage
- Content manipulation
- Platform takedown

## Mitigations
- End-to-end encryption
- Multi-path routing and fallback
- Decentralized storage integration
- Anonymous posting with moderation
