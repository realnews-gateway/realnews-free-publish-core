# Module Overview

## VPN Access Layer
Provides anti-censorship network access using modern protocols (QUIC, WebSocket, TLS obfuscation).

## News Aggregation
Aggregates RSS feeds and mirrors independent media sources for resilient access.

## Lightweight BBS
Anonymous discussion board with moderation tools and lightweight storage.

## Secure Uploader
Allows users to safely transmit text, images, and video to external platforms or decentralized storage (IPFS, Arweave).
