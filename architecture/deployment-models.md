# Deployment Models

RealNews Free Publish Core supports multiple deployment models based on risk level and operational capacity:

## 1. NGO / Media Deployment

- Full system deployment  
- Multi‑node mirroring  
- Secure uploader integration  
- Public mirror site hosting  
- Offline bundle generation

## 2. Individual Activist Deployment

- Emergency Publishing Channel only  
- Lightweight client  
- Encrypted submission  
- Decentralized storage only

## 3. Community Relay Deployment

- Mirror node only  
- No submission interface  
- Automatic sync from trusted sources  
- Optional offline distribution

All deployments are modular, resource‑adaptive, and designed for censorship‑resistant environments.
